# Brand guide (placeholder)

This folder is a placeholder for the Figma brand and prototype required by the course.

What to include in your Figma file:
- A cover/artboard with logo and color palette (primary, secondary, accent)
- Typography scale and examples (H1, H2, body)
- At least 4 artboards representing pages: Home, About, Work, Contact
- Clickable prototype showing navigation between pages

Suggested palette used in this project:
- Primary: #0B72B9
- Accent: #FFB020
- Neutral dark: #1B2430
- Light background: #F3F8FC

Exporting to this repo:
1. In Figma, create artboards for each page.
2. Export each artboard as PNG and add it to this folder (e.g. `home.png`).
